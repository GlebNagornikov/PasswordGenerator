# PasswordGenerator
Простейший генератор паролей. Позволяет задавать длину, спецсимволы, русский/английский алфавит, цифры и регистр. 
Среди дополнительных возможностей проверка пароля на соответствие заданным требованиям и исключение выбранных символов из генерации.
